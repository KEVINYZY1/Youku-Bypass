简单说明
---
直接绕过所有限制，无缝播放。程序会自动选择最高清画质的视频

产生aria2c下载脚本
---
flvxz.pl -aria2c 'http://v.youku.com/v_show/id_XNDM0MjUwMjQ0.html'

直接在线播放
---
flvxz.pl -mplayer 'http://v.youku.com/v_show/id_XNDM0MjUwMjQ0.html'
